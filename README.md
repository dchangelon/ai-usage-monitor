# AI Usage Monitor

A desktop application for tracking AI API usage and costs across providers. Built with Tauri and React.

## Status

Work in progress.

## Tech Stack

- **Desktop Framework**: Tauri 2
- **Frontend**: React 19, Vite
- **Database**: SQLite (via Tauri)

## Getting Started

### Prerequisites

- Node.js 20+
- Rust (for Tauri)

### Development

```bash
npm install
npm run tauri dev
```

### Build

```bash
npm run tauri build
```
